# 03｜竞品逆向分析（product-reverse-analysis）

> 5 步分析框架，从产品截图逆向推导业务逻辑、架构和交互设计。

## 核心概念

### 5 步分析框架

截图输入 → 视觉分析 → 交互分析 → IA 分析 → 输出 Mermaid 流程图

### 三维度分析

| 维度 | 分析内容 | 输出格式 |
|------|---------|---------|
| 视觉 | 配色、字体、布局、组件 | 视觉规范表 |
| 交互 | 操作流程、反馈机制、状态转换 | 流程图 |
| IA | 信息层级、导航结构、内容组织 | 站点地图 |

### Mermaid 流程图

- **flowchart** — 用户操作路径
- **sequenceDiagram** — 页面/组件间交互时序

## 竞赛价值

- 展示「产品截图 → 完整分析报告」的自动化能力
- 适合 Code With SOLO 赛道，技术向效果好
