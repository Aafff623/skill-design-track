# 01｜Skill 概念与文件结构

> Skill 是一个结构化的 AI 工作流配置包，包含主指令、模板、参考资料和辅助脚本。

## 核心概念

### 什么是 Skill

Skill 是一个可复用的「工作流单元」，像代码函数一样有输入/输出/逻辑：

| 组成 | 作用 |
|------|------|
| SKILL.md | 主指令文件，定义触发场景和工作流 |
| assets/ | 模板文件，如输出格式模板 |
| references/ | 参考资料，如分类指南、识别模式 |
| scripts/ | 辅助脚本，如 CSV 读取工具 |

### Skill 的触发机制

description 字段决定何时自动触发。好的 description 要列举：
- 典型用户语言
- 适用场景
- 关键词变体

描述越精准，触发越准确。

## 设计原则

1. **触发精准** — description 列举典型用语、适用场景、关键词变体
2. **人在环路中** — 关键决策点让用户确认，而非全自动
3. **强指令词** — 必须/始终/严禁 比 尽量/最好 执行力更高
4. **结构分离** — SKILL.md 只写做什么，模板抽到 assets/，规则抽到 references/

## 在大赛中的位置

这门课直接对应 SOLO 技能创作赛的「创作」赛道。每个 Skill 都可作为参赛作品原型。
